# Raou Travel - Clean Distribution

This is the lightweight version of the Raou Travel project, optimized for sharing and distribution.

## How to Run
1.  **Double-click `START-APP.command`**.
2.  The first time you run it, it will automatically download and install the necessary tools (Node.js and dependencies). This may take a few minutes.
3.  A popup will appear asking if you want to run the **Website** or the **Editor**.
4.  Make your selection, and the app will launch in your browser.

## Features
-   **Lightweight**: < 10MB (excluding dependencies).
-   **Portable**: Runs on any Mac, even without Node.js installed (it installs a portable version if needed).
-   **Unified**: One script handles everything.
-   **SEO Optimized**: Includes `react-helmet-async` for better search engine visibility.

## For Developers
-   This folder contains the full source code in `src`.
-   To build for production manually: `npm run build`.
-   To start the server manually: `npm run dev:all`.
